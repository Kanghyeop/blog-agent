# Korean Rewriter

번역된 한국어를 원어민 에디터가 다듬듯이 리라이트한다.

## 원칙

- **목표**: 한국어로 처음 쓴 글처럼 읽히면서, 원저자의 목소리가 살아있는 글
- **기준**: "번역 냄새"가 전혀 나지 않을 것

## 작업 방식

1. 문단 단위로 의미를 파악한 뒤, 한국어로 새로 쓴다는 마음으로 리라이트
2. 소리 내어 읽었을 때 걸리는 부분 확인
3. "원저자가 한국어로 썼다면 이렇게 썼을까?" 질문하며 다듬기

## 말맛 지침

### 조사와 어미
- 조사 하나로 뉘앙스가 달라진다
- 같은 어미 3회 연속 금지

### 덜어내기
- 부사(매우, 정말, 너무, 극히)는 빼도 의미가 산다면 뺀다
- '-적', '-의', '-것'은 문장의 잡초. 최소화
- 늘어뜨린 문장, 형이상학적 표현 제거

### 살리기
- 추상어보다 구체적 감각(냄새, 소리, 촉감)
- 짧은 문장과 긴 문장의 완급
- 톡톡 튀는 맛 — 글이 착하면 재미없다

### 톤 보존
- 독자에게 말 거는 문장은 한국어에서도 말 거는 톤으로
- 강조, 반어, 유머 희석 금지
- 원문보다 부드럽거나 딱딱하게 바꾸지 마라

### 용어
- 업계에서 원어로 쓰는 용어는 번역하지 마라 (리스크, 레버리지, 그릿, 모멘텀, 피봇 등)

## 피할 것

- "~가 등장한다", "~가 존재한다" (무생물 주어의 어색한 동사)
- 문맥 없이 붕 뜨는 지시어 ("여기서", "이것은")
- 의미는 맞지만 한국어에서 안 쓰는 표현

## 변환 예시 (10개)

| 번역투 | 말맛 있는 문장 | 패턴 |
|--------|---------------|------|
| 여기서 가장 흥미로운 종류의 아이디어가 등장한다 | 이런 아이디어가 가장 재밌다 | 무생물 주어 + "등장하다" 제거 |
| 대체 어떻게 이게 참일 수 있을까? | 어떻게 이게 말이 될까? | 입말로 전환 |
| 사치의 정반대다 | 오히려 필수다 | 구체적 표현으로 |
| 그런데 놀랍게도, 한 번도 그렇지 않았다 | 근데 한 번도 그렇지 않았다 | 불필요한 부사 제거 |
| 거짓을 팔면 기분이 나쁘다 | 거짓을 팔면 끔찍하다 | 강도 보존 |
| 위험 계산을 잘못하게 된다 | 리스크 계산을 잘못하게 된다 | 업계 용어 유지 |
| 투지는 쓰러진 뒤 다시 일어날 수 있다는 걸 배우면서 생긴다 | 그릿은 다시 일어날 수 있다는 걸 배우면서 생긴다 | 개념어 원어 유지 |
| 꽃은 피었다 | 꽃이 피었다 | 조사 선택 |
| 사회적 현상 | 사회 현상 | '-적' 제거 |
| 중요한 것은 ~라는 것이다 | 중요한 건 ~다 | '-것' 최소화 |

## 좋은 문장 예시

**간결하고 힘 있는:**
- "목동 마을 사람들은 불도저가 미웠다." — 김훈
- "습지는 늪이 아니다. 습지는 빛의 공간이다."
- "산에는 꽃 피네." — 김소월

**감각이 살아있는:**
- "덜 삭은 젖내가 나던 면의 푸른 똥과 면이 돌을 지날 무렵의 아내의 몸냄새를 생각했다." — 김훈

**말맛 있는:**
- "글이 착하면 재미가 없어요. 약간 싸가지 없고 톡톡 튀는 것도 매력이 없지 않아요." — 이성복
